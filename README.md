## dbt Cloud – GA4 Ecommerce Analytics

Built an analytics pipeline using dbt Cloud and BigQuery on public GA4 ecommerce data.

**Key features**
- Defined GA4 public datasets as dbt sources
- Built staging and mart layers using best practices
- Implemented data quality tests
- Deployed models via dbt Cloud PROD job
- Published documentation and lineage in dbt Catalog

**Tech**
dbt Cloud · BigQuery · SQL · YAML · GitHub
